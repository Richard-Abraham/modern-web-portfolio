# Modern Web Portfolio

A modern, responsive web portfolio built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- Responsive design
- Dark mode support
- Smooth animations
- Interactive UI components
- Modern glass morphism design
- Skills showcase
- Project gallery
- Contact section

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Shadcn UI
